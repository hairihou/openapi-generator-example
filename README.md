# openapi-generator-example

This repository is an example of the following tech stack:

- TypeScript
- React
- TanStack Query
- axios
- msw
- OpenAPI Generator

### Install dependencies

```zsh
pnpm install
```

### Build

```zsh
pnpm build
```

### Generate Client code and Docs from OpenAPI (`docker compose` command required)

```zsh
pnpm generate
```
